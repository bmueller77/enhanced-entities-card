# Enhanced Entities Card

An enhanced version of the stock Lovelace **Entities** card:

- static and **state‑dependent** icon colors and icons (true/false maps or numeric
  thresholds)
- optional **color gradient** between thresholds
- **divider** and **markdown** rows (with live `{iobroker.id}` templates)
- per‑row **tap action** (`more-info` / `toggle` / `none`), `hide_state`,
  `icon_position: right`
- fully adjustable spacing (font size, row gap, paddings, icon size, icon gap)
- a complete **visual GUI editor**

See the [README](README.md) for full configuration and examples.
